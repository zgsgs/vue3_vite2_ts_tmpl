# 下一代 API 模拟

## 使用范围

给日常开发 mock 数据时使用，不是给 storybook 使用的。只是日常开发和 storybook 可以共用 MSW 工具。

## 功能

通过拦截网络级别的请求来模拟。无缝重用相同的模拟定义进行测试、开发和调试。
