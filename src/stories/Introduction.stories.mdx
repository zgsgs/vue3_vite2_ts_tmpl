# 欢迎来到Storybook

Storybook 可帮助您独立于应用程序的业务逻辑、数据和上下文构建 UI 组件。
这使得开发难以到达的状态变得容易。将这些 UI 状态保存为 **stories**，以便在开发、测试或 QA 期间重新访问。

现在通过在侧栏中导航到示例故事来浏览示例故事。在 `src/stories` 目录中查看它们的代码以了解它们是如何工作的。
我们建议使用 [**component-driven**](https://componentdriven.org) 流程构建 UI，从原子组件开始，以页面结束。
